# 软件测试学习平台

一个现代化、专业的软件测试学习网站，专为展示软件测试知识体系和个人技能而设计。

## 🌟 特点

- **多页面架构**：首页、关于我、学习路径、项目展示、联系方式独立页面
- **侧边栏导航**：学习路径页面采用左侧导航+右侧内容的文档式布局
- **现代化设计**：渐变色、卡片式布局、流畅动画效果
- **响应式设计**：完美适配桌面端和移动端
- **详细内容**：涵盖软件测试基础、功能测试、接口测试、自动化测试、性能测试、进阶技能

## 📁 项目结构

```
├── index.html          # 首页
├── about.html          # 关于我
├── learning.html       # 学习路径（核心页面）
├── projects.html       # 项目展示
├── contact.html        # 联系方式
├── styles/
│   ├── main.css       # 全局样式
│   ├── header.css     # 导航栏样式
│   ├── learning.css   # 学习路径页面样式
│   ├── about.css      # 关于页面样式
│   ├── projects.css   # 项目页面样式
│   └── footer.css     # 页脚和联系页面样式
└── scripts/
    ├── main.js        # 全局脚本
    └── learning.js    # 学习路径页面交互脚本
```

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 即可访问网站
2. 无需安装任何依赖，纯HTML/CSS/JavaScript实现
3. 可以部署到任何静态网站托管服务（GitHub Pages、Netlify等）

## 💡 自定义

### 修改个人信息
- 在 `about.html` 中修改个人简介和技能栈
- 在 `contact.html` 中更新联系方式
- 在 `projects.html` 中添加你的项目案例

### 修改颜色主题
在 `styles/main.css` 中修改CSS变量：
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #7c3aed;
    --accent-color: #06b6d4;
}
```
